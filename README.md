# linktious

## How to start

Clone project using:

```bash
git clone git@github.com:Linktious/linktious.git
cd linktious
make
```

Compose project in dev mode using:

```bash
docker-compose -f "docker-compose.debug.yml" up -d --build
```

It is recommended to use `.linktiousrc` shortcuts
```bash
source ./.linktiousrc
```
